Исправление топов по именам:
- имя теперь сохраняется не только на входе, но и при любой продаже
- TopCache при refresh дополнительно пытается зафиксировать известные имена
- Placeholder при запросе имени сначала берет онлайн-игрока, затем OfflinePlayer, затем player-data.yml
- при старте выполняется backfill player_names для UUID из profession sections

Важно: после запуска дайте игроку один раз зайти на сервер или что-то продать, затем перезагрузите голограммы.
